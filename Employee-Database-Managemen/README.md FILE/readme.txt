# Employee Database Management System

This project is an SQL-based employee database management system. It demonstrates:
- Creating tables for managing employee and department data.
- Inserting employee and department data into tables.
- Writing SQL queries for employee data retrieval and analysis.

## Project Structure
- **tables.sql**: Contains SQL statements to create the database and tables.
- **insert_data.sql**: Inserts sample data into the tables.
- **queries.sql**: Includes various SQL queries for data analysis and retrieval.

## Technologies Used
- SQL (MySQL/PostgreSQL)
